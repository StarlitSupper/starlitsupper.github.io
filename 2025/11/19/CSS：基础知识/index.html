HTML/CSS前端

CSS用于给HTML结构设置样式：

  • 层叠样式表（Cascading Style Sheets）
  • 它也是一种标记语言

优先级：行内样式>内部样式=外部样式

语法规范

CSS规则由两部分构成：

  • 选择器：找到需添加样式的元素
  • 声明块：设置具体的样式，声明的格式为name: value;

选择器优先级

行内 > id 选择器 > 类选择器 > 元素选择器 > 通配选择器

!important最超模，其他情况下，高层级优先，同层级依次向下比，全部等同的情况下看先后顺序。

其实鼠标在 VS Code 里悬停一下能看。

CSS特性

  1. 层叠性：发生样式冲突时，会根据选择器优先级，进行样式的层叠（其实就是覆盖）
  2. 继承性：元素会自动拥有父元素或者祖先元素设置的某些样式（text-…、font-…、line-…、color等），优先继承近的
  3. 优先级：这是一个权重的问题
